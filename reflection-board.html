ふりかえり見取りボード

学習計画表を一括で読み取り、振り返りの見取りと明日の声かけにつなぐ（教師用）

教師の見取り用ボードです。記述はこの端末の中だけで処理され、どこにも送信されません。
くわしくS〜Dの段階は指導の手がかりであり、成績の評定ではありません。子どもに返す・掲示するものではありません。点数・順位・学級平均は出しません（くらべる相手は前回のじぶんだけ）。

  2. 2表を読み込む
  3. 3結果を見る
  4. 4書き出す

4書き出し

1一人分の学習計画表の読み込み

写真・PDF読み込みの設定と注意（AIの種類・APIキー）　※読み込む前に必ず読んでください
写真・PDFの読み込みは、画像を外部AI（Gemini または Claude）に送信して文字に起こします。
・氏名・学校名が写っている部分は、隠すか切り取ってから読み込んでください（送信前に毎回確認します）
・子どもの記述をAIが読むことになります。学校・保護者の方針、自治体の生成AI利用ガイドラインに従ってお使いください
・Geminiは無料のAPIキー（Google AI Studio）でも使えます。ただし無料枠では、送信した内容がモデルの改善に使われる場合があります（有料キーは対象外）。学校利用では有料キーまたはClaudeをおすすめします
・キーはこの端末にのみ保存され、文字起こし以外には使いません。表の貼り付け・ファイル読み込み・判定では一切通信しません

APIキーの取得（コピーして上の欄に貼り付けます）：
・Gemini：Google AI Studio − APIキー（Googleアカウントで無料から作成できます）
・Claude：Anthropic Console − API Keys（要アカウント・有料）
