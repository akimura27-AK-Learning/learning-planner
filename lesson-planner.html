STEP 1授業の基本情報

教科・単元・本時の目標を入力します。ここで入れた内容は、後のAI相談と展開表に引き継がれます。

STEP 2本時の展開と方略の埋め込み

導入・展開・まとめの各段階に、学習活動と調整方略を入れていきます。「＋方略を追加」から3層（I/You/We）の方略を選べます。AIに相談しながら埋めることもできます。

既存の指導案（Word / PDF）を読み込む手元の学習指導案を取り込んで、各段階の活動に振り分けられます
方法A：AIで自動振り分け
読み取った内容を、導入・展開・まとめの活動枠にAIが整理して入れます（要・AI設定）。
方法B：手作業で貼る
下の「読み取ったテキスト」から必要な部分をコピーして、各活動欄に自分で貼り付けます。

STEP 3AIと相談してブラッシュアップ

まず学習活動そのものを具体的にし、活動が固まってから調整方略を入れていきます。下のタブで相談のモードを切り替えられます。AIが具体的な学習活動や調整方略を提案すると、「＋ 指導案に追加」ボタンが付き、単元構想に反映できます。

STEP 4本時の指導案

Wordファイルでの保存、印刷／PDF保存ができます。修正は前の画面に戻って行えます。

SETUPAI接続の設定

このツールのAI相談を使うには、あなた自身のAPIキーが必要です。下の手順でキーを取得し、入力してください。キーはこのブラウザの中だけで使われ、ファイルや外部には保存されません（ページを再読み込みすると消えます）。キーは料金に直結するので、他の人に教えないでください。
初めてなら Gemini がおすすめです。Googleアカウントがあれば無料枠で使い始められ、クレジットカードの登録も不要です。
② Claude のキーを取得する手順
  1. console.anthropic.com を開き、Anthropicアカウントでログインします（なければ新規登録）。
    → キー発行ページを開く
  2. 支払い方法（クレジットカード）の登録が必要です。使った分だけの従量課金です。
  3. 左メニューの Settings → API keys を開き、Create Key を押します。
  4. 名前（例：指導案ツール）を付けて作成し、表示された sk-ant-… で始まる文字列をコピーします。
    ※キーは作成時の一度しか表示されません。必ずその場でコピーしてください。
  5. 下の入力欄に貼り付けます。
② Gemini のキーを取得する手順
  1. Google AI Studio を開き、Googleアカウントでlog inします。
    → キー発行ページを開く
  2. 「Create API key（APIキーを作成）」ボタンを押します。
  3. プロジェクトの選択を求められたら、案内に従って進めます（既存のものでOK）。
  4. 表示された AIza… で始まる文字列をコピーします。
  5. 下の入力欄に貼り付けます。
    ※無料枠の範囲ならカード登録なしで使えます。
② OpenAI (GPT) のキーを取得する手順
  1. platform.openai.com を開き、OpenAIアカウントでログインします（なければ新規登録）。
    → キー発行ページを開く
  2. 支払い方法（クレジットカード）の登録が必要です。無料枠は基本的になく、使った分だけの従量課金です。
  3. API keys のページで Create new secret key を押します。
  4. 名前を付けて作成し、表示された sk-… で始まる文字列をコピーします。
    ※キーは作成時の一度しか表示されません。必ずその場でコピーしてください。
  5. 下の入力欄に貼り付けます。
「✓ 接続できました」と出れば準備完了です。エラーが出たら、キーの貼り間違いがないか確認してください。
